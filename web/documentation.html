API Documentation

Create Collection

To create a new collection in Igor Database, use the following endpoint:

POST /create_collection

Body:

{
    "collection_name": "collection_name"
}

Insert Document

To insert a new document into a collection, use the following endpoint:

POST /insert_document

Body:

{
    "collection_name": "collection_name",
    "document": {
        "key": "value",
        ...
    }
}

Find Document

To find documents in a collection that match a query, use the following endpoint:

POST /find_document

Body:

{
    "collection_name": "collection_name",
    "query": {
        "key": "value",
        ...
    }
}
