API Documentation

Create Collection

**POST** `/create_collection`

{
    "collection_name": "collection_name"
}

Insert Document

**POST** `/insert_document`

{
    "collection_name": "collection_name",
    "document": {
        "key": "value",
        ...
    }
}

Find Document

**POST** `/find_document`

{
    "collection_name": "collection_name",
    "query": {
        "key": "value",
        ...
    }
}
