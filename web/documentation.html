Sobre

Igor Database é um banco de dados NoSQL simples que armazena dados em arquivos de texto e pode ser gerenciado via terminal. Este documento fornece instruções sobre como configurar, usar e atualizar o Igor Database.

Endpoints da API

Criar uma Coleção

curl -X POST http://127.0.0.1:5000/create_collection -H "Content-Type: application/json" -d '{"collection_name": "my_collection"}'

Inserir um Documento

curl -X POST http://127.0.0.1:5000/insert_document -H "Content-Type: application/json" -d '{"collection_name": "my_collection", "document": {"name": "Alice", "age": 25}}'

Encontrar Documentos

curl -X POST http://127.0.0.1:5000/find_documents -H "Content-Type: application/json" -d '{"collection_name": "my_collection", "query": {"name": "Alice"}}'

Listar Coleções

curl http://127.0.0.1:5000/list_collections

Atualizar um Documento

curl -X POST http://127.0.0.1:5000/update_document -H "Content-Type: application/json" -d '{"collection_name": "my_collection", "query": {"name": "Alice"}, "update": {"age": 26}}'

Deletar um Documento

curl -X POST http://127.0.0.1:5000/delete_document -H "Content-Type: application/json" -d '{"collection_name": "my_collection", "query": {"name": "Alice"}}'

Deletar Todos os Documentos de uma Coleção

curl -X POST http://127.0.0.1:5000/delete_all_documents -H "Content-Type: application/json" -d '{"collection_name": "my_collection"}'

Instruções de Execução

1. Configurar o Ambiente

# Clone o repositório
git clone https://github.com/seu_usuario/igor-db.git

# Navegue até o diretório do projeto
cd igor-db

# Crie e ative um ambiente virtual
python -m venv venv
source venv/bin/activate  # No Windows use: venv\Scripts\activate

# Instale as dependências
pip install -r requirements.txt

2. Iniciar o Banco de Dados

# Execute o servidor Flask
python -m igor.api

3. Parar o Banco de Dados

# Para parar o servidor Flask, você pode usar CTRL+C no terminal onde o servidor está rodando.

4. Usar a Aplicação

Após iniciar o banco de dados, você pode usar os endpoints da API listados acima para interagir com o banco de dados Igor. Use os exemplos de `curl` fornecidos para testar as funcionalidades.
